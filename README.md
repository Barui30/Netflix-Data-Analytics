# Netflix Data Analytics Project

## 📌 Overview
This project analyzes Netflix Movies and TV Shows using descriptive data analytics.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- VS Code

## 📊 Analysis Performed
- Movies vs TV Shows distribution
- Year-wise content trend
- Country-wise analysis
- Genre analysis
- Ratings distribution
- Duration analysis

## 🔍 Key Insights
- Netflix content increased rapidly after 2015
- USA produces the most content
- Drama is the most popular genre
- Most content is targeted towards mature audiences
